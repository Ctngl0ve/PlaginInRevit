# Плагин под программу Revit2021
## Создание "ленты" - Application
Код реализует возможность использовать ленту с именем "Организация" на панели выбора инстументов в программе
## Ссылка на команду - Command
Код ссылается на плагин сборника плагинов *"Revit2019SDK"* -  __DeleteDimensions__

Плагин позволяет удалять незакрепленные размеры путем выделения их и активировании плагина
